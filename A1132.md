# A1132

## 1.题意理解
判断一个整数从中间位数截断后，前半部分能否整除后半部分。

## 2.思路分析
按照要求实现即可，注意判断分母为0情况。

## 3.参考代码
```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
    int n;
    string s;

    scanf("%d", &n);
    for(int i = 0; i < n; i++)
    {
        cin >> s;
        int l = s.length();
        string t1 = s.substr(0, l / 2);
        string t2 = s.substr(l / 2, l / 2);

        if(stoi(t1) * stoi(t2) == 0)
            printf("No\n");
        else
        {
            if(stoi(s) % (stoi(t1) * stoi(t2)) == 0)
                printf("Yes\n");
            else
                printf("No\n");
        }
    }

    return 0;
}
```
